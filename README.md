Добро пожаловать в README файл!

Для запуска калькулятора и юнит тестов поместите оба файла в единый каталог. Затем при помощи любого интерпретатор языка запустите программу.

calculator.py - файл с функциями калькулятора разнесёнными по модулям
test_calculator.py - файл с юнит тестами функций калькулятора (позитивные и негативные проверки)
